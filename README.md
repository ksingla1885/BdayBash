# 🎉 Birthday Wish Generator

A beautiful, personalized birthday website generator that creates an emotional, animated surprise link for your loved ones. 

## Features
- **✨ Core Wish Creation:** Submit a personalized birthday wish with receiver/sender names.
- **🖼️ Memory Timeline:** Upload multiple images that act as a sliding timeline backdrop.
- **🤖 AI Messages:** Toggle between emotional, funny, or savage messages generated by AI based on tone.
- **💥 Visuals:** Stunning UI with Framer Motion, typing text animations, and confetti bursts.
- **🔒 Backend API:** Node/Express/Mongoose back-end with Cloudinary image hosting.

## Built With
* Frontend: React (Vite), Tailwind CSS, Framer Motion, canvas-confetti, Axios
* Backend: Node.js, Express, MongoDB
* Tech/Services: Cloudinary, OpenAI

## Getting Started

1. **Setup MongoDB & Cloudinary & OpenAI**
   - Get your MongoDB connection string.
   - Set up Cloudinary account for Image hosting.
   - Get OPENAI_API_KEY.
   - Add these credentials in `backend/.env` file.

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. Go to `http://localhost:5173` and start generating! 🎁

## 🚀 Deployment (Vercel)

For a monorepo setup, it's best to deploy **Frontend** and **Backend** as two separate Vercel projects:

### 1. Backend (API)
- **Framework Preset**: Node.js
- **Root Directory**: `backend`
- **Environment Variables**:
  - `MONGO_URI`: *Your MongoDB connection string*
  - `CLOUDINARY_CLOUD_NAME`: *Your Cloudinary cloud name*
  - `CLOUDINARY_API_KEY`: *Your Cloudinary API key*
  - `CLOUDINARY_API_SECRET`: *Your Cloudinary API secret*
  - `GROQ_API_KEY`: *Your Groq API key*
  - `CLIENT_URL`: *The URL of your deployed frontend*
  - `NODE_ENV`: `production`

### 2. Frontend
- **Framework Preset**: Vite
- **Root Directory**: `frontend`
- **Environment Variables**:
  - `VITE_API_BASE_URL`: *The URL of your deployed backend* (e.g. `https://your-api.vercel.app/api`)
